# P4Projekt2

This is a simple **chat application** with a **.NET MAUI frontend** and an **ASP.NET backend**.

## Features

- User registration and login
- Adding and managing friends
- Manual chat refresh via a refresh button
- Integrated **chatbot** that responds to user messages
- Chatbot powered by a **GPT-2 community model** from Hugging Face

## Technologies Used

- **.NET MAUI** – Cross-platform frontend for mobile and desktop
- **ASP.NET Core** – Backend REST API
- **Entity Framework Core** – For database interactions
- **JWT (JSON Web Tokens)** – Authentication and session handling
- **GPT-2** – Integrated language model for chatbot responses (self-hosted using Hugging Face Transformers)
